# iPhone 照片查看及整理应用

基于SwiftUI开发的智能照片整理应用，专注于照片自动分组和智能推荐。

## 核心功能

### 1. 同场景自动分组
- **时间接近性分组**：30秒内连续拍摄的照片
- **地点接近性分组**：50米范围内拍摄的照片
- **视觉相似度分组**：画面内容相似的照片（使用Vision框架）
- **典型场景**：连拍、同一机位多拍、同一时刻连续调整构图

### 2. 分层推荐（4个层级）
- **主推荐**：综合表现最佳的照片（1张）
- **建议保留**：质量不错，值得保留（1-3张）
- **可选保留**：可根据需要选择保留
- **建议删除**：可考虑删除以节省空间

### 3. 推荐解释
每张照片显示推荐理由：
- 最清晰
- 主体更完整
- 人物睁眼
- 构图更稳定
- 与其他照片高度重复

### 4. 用户确认后处理
- **一键接受推荐**：自动选择推荐结果
- **手动改选**：灵活调整选择
- **批量删除未保留项**：明确提示"将移入系统最近删除"

## 技术架构

### 技术栈
- **UI框架**：SwiftUI（声明式界面）
- **照片访问**：Photos Framework + PhotoKit
- **图像分析**：Vision Framework（特征提取、人脸检测）
- **地理位置**：Core Location
- **数据存储**：Core Data（用户选择记录）
- **并发处理**：Async/Await + OperationQueue

## 评分算法

照片评分基于多个维度加权计算：

| 维度 | 权重 | 检测方法 |
|------|------|----------|
| 清晰度 | 25% | 拉普拉斯方差（图像模糊检测） |
| 主体完整性 | 20% | Vision人物/主体检测边界框覆盖率 |
| 人物状态 | 15% | 人脸检测 + 眼睛开闭状态 |
| 构图稳定性 | 15% | 水平线倾斜角度 + 主体居中程度 |
| 重复度 | 25% | 与组内其他照片的视觉相似度 |

## 安装与运行

### 环境要求
- macOS 13.0 或更高版本
- Xcode 15.0 或更高版本
- iOS 16.0 或更高版本（目标设备）

### 权限要求
在 `Info.plist` 中添加以下权限描述：
- `NSPhotoLibraryUsageDescription` - 访问照片库
- `NSPhotoLibraryAddUsageDescription` - 添加照片到照片库
- `NSLocationWhenInUseUsageDescription` - 访问地理位置（可选）

### 构建步骤
1. 使用Xcode打开项目
2. 选择目标设备（iPhone 模拟器或真机）
3. 配置开发者证书
4. 构建并运行应用

## 性能优化

### 内存管理
- 使用`PHImageManager`分级加载图片（缩略图→中等尺寸→原图）
- 批量处理照片，避免同时加载大量高分辨率图片
- 及时释放`CGImage`、`CIImage`等大内存对象

### 后台处理
- 使用`OperationQueue`控制并发数
- 分析过程支持暂停/继续
- 进度反馈和预估时间显示

### 缓存策略
- 特征向量缓存到本地文件
- 评分结果缓存到Core Data
- 缩略图使用`PHCachingImageManager`

## 隐私保护

### 设备端处理
- 所有分析都在设备端完成
- 不上传照片到云端
- 不收集个人身份信息

### 权限控制
- 明确的权限请求说明
- 支持受限照片库访问（iOS 14+）
- 用户可以随时撤销权限

### 数据安全
- 用户选择记录本地加密存储
- 支持隐藏/加密相册功能
- 明确的删除确认提示

## 使用流程

1. **照片导入**：自动扫描照片库，分析照片元数据
2. **智能分组**：按时间、地点、视觉相似度自动分组
3. **质量评估**：为每组照片计算综合评分
4. **分层推荐**：生成主推荐、建议保留、可选保留、建议删除
5. **用户确认**：一键接受推荐或手动调整选择
6. **批量处理**：确认删除未保留照片，明确提示操作后果
7. **结果查看**：显示清理统计和空间节省情况

## 可配置参数

### 分组参数
- 时间分组阈值：5-120秒
- 地点分组阈值：10-200米
- 视觉相似度阈值：70-95%

### 推荐参数
- 自动接受推荐：开关
- 显示详细解释：开关
- 保留原始照片：开关

- **JSON格式**：适合程序读取和交换数据
- **CSV格式**：适合在表格软件中查看
